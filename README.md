ATS Yard Operations Center
A yard operations management prototype built for Anderson Trucking Service (ATS) — Jacksonville yard.
Overview
A React/Next.js dashboard designed to replace Monday.com for day-to-day yard operations. Pre-loaded with live fleet data and built to manage trailer tracking, inbound/outbound loads, Puerto Rico port shipments, mechanic red-tags, and local driver assignments.
Tech Stack

Next.js / React
TypeScript
Tailwind CSS

Status
Prototype — currently in review as a potential internal operations tool.
Getting Started
bash 
npm install
npm run dev
